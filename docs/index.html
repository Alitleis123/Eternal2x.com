What is Eternal2x ?

Eternal2x is a plugin for DaVinci Resolve that improves the quality and smoothness of clips with intelligent motion-aware upscaling — similar to what Topaz and Flowframes do, but built right into your timeline.

How it works

4 buttons. 1 slider. That's the entire UI.

⏱ Detect Motion

Analyzes every frame using tile-based motion scoring. Places labeled markers on your timeline so you can preview and fine-tune exactly where motion happens.

✂ Sequence

Cuts at every marker position and converts clips into 1-frame segments for precise interpolation control. Your markers are the source of truth.

▦ Regroup

Removes gaps left over from cutting and makes the entire sequence continuous again. Markers shift to match the new positions automatically.

⚡ Upscale + Interpolate

Applies 2x Super Scale to every clip. Motion segments get Optical Flow interpolation. Static segments use Nearest for speed. Smart, not brute-force.

Why Eternal2x?

🔌 Built into Resolve

No external software, no render-export-reimport loops. Eternal2x runs as a native Resolve script panel directly on your timeline.

🔨 One-Click Install

Double-click the installer — it handles Python, packages, and Resolve setup automatically. No terminal commands, no manual file copying.

🔄 Auto Updates

Install once, stay current forever. Eternal2x checks for updates every time you open it and applies them silently in the background.

🎨 Motion-Aware

Only interpolates where it matters. Static areas stay fast. Motion areas get full Optical Flow treatment. You control the sensitivity.

Requirements

DaVinci Resolve 18+ (Free or Studio)

Windows 10/11 or macOS 12+

Python is handled automatically by the installer — you don't need to install it yourself.
